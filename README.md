Greentick
=========

Check latest software version using batch script. Currently supported software are:
* Chrome
* Firefox
* Java

### Basic usage

run fetch.bat
```batch
fetch.bat
```

output:

```json
{
  "chrome": "36.0.1985.143",
  "firefox": "31.0",
  "java": "7.0.670.1"
}
```
